# gestora-de-moda-feminina
I é um projeto de chatbot desenvolvido com foco em personalização e aprendizado contínuo. Criado usando tecnologias de processamento de linguagem natural (NLP), este bot é capaz de manter diálogos contextuais, responder perguntas, fazer atendimento e gerar suporte em pagamentos, devoluções ou troca e forma de envio.
